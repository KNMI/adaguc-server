NCWMSExtensions
===============

ADAGUC has support for some of the WMS extensions as built in into
ncWMS.
See their page for details:
http://www.resc.rdg.ac.uk/trac/ncWMS/wiki/WmsExtensions

ADAGUC supports the following ncwms extensions:

-   COLORSCALERANGE
-   OPACITY
-   NUMBANDS
-   LOGSCALE
